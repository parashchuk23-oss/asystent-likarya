# Асистент лікаря

## Опис

Асистент лікаря — веб-проєкт для медичних працівників України.

Мета продукту — зібрати в одному місці щоденні клінічні інструменти лікаря:

* калькулятори;
* опитувальники;
* оцінку ризиків;
* форму для структурованого лікарського заключення;
* у майбутньому — AI-допомогу для генерації заключень.

## Цільова аудиторія

* сімейні лікарі
* кардіологи
* терапевти
* ендокринологи
* інтерни
* лікарі загальної практики

## Поточний формат

Проєкт реалізовано як веб-сайт на Next.js.

Desktop/Electron-напрямок не продовжується.

## Поточний стек

* Next.js
* React
* Tailwind CSS
* Vercel
* GitHub

## Основні модулі

* Асистент лікаря
* Кардіоваскулярний ризик
* Ниркова функція
* Калькулятори
* Опитувальники

## Принципи

* простота;
* швидкість роботи;
* доказова медицина;
* мобільна доступність;
* мінімум зайвих кліків;
* не перевантажувати інтерфейс;
* не додавати авторизацію, базу чи оплату до появи реальної потреби.

## Медичний дисклеймер

Асистент лікаря є допоміжним інструментом для медичних працівників.

Результати калькуляторів, опитувальників та майбутні AI-згенеровані тексти не замінюють клінічне рішення лікаря.
