# Roadmap

## Етап завершено

### Запуск сайту

* Створено веб-проєкт на Next.js.
* Реалізовано головну сторінку з вкладками.
* Додано базові клінічні інструменти.

### Деплой

* Сайт опубліковано на Vercel.
* GitHub підключено як джерело коду.
* Production alias налаштовано для `asystent-likarya.vercel.app`.

### SEO база

* Додано title та description.
* Додано Google Search Console verification.
* Додано sitemap.xml.
* Додано robots.txt.

### Логотип

* Обрано Modular A.
* Створено SVG та PNG assets.
* Додано favicon.
* Додано логотип у header.

### Брендбук

* Створено `BRANDBOOK.md`.
* Зафіксовано позиціонування, кольори, типографіку та візуальні принципи.

## Наступний етап

* Створити окремі SEO-сторінки для ключових калькуляторів.
* Додати сторінки з поясненням шкал і клінічного використання.

### Контент

* Підготувати короткі медичні описи для калькуляторів.
* Підготувати статті для органічного трафіку.
* Додати структуровані FAQ-блоки.

### Просування

* Перевірити індексацію в Google Search Console.
* Додати сайт у професійні спільноти лікарів.
* Зібрати перший фідбек від лікарів.

Повернутися до цього після:

* стабілізації UI;
