# Асистент лікаря

Next.js веб-застосунок для медичних працівників: асистент лікаря, кардіоваскулярний ризик, ниркова функція, калькулятори та опитувальники.

## Можливості

- Асистент лікаря
- Кардіоваскулярний ризик
- Ниркова функція
- Калькулятори
- Опитувальники

## Встановлення залежностей

```bash
npm install
```

## Локальний запуск

```bash
npm run dev
```

Після запуску відкрити:

```text
http://localhost:3000
```

Якщо порт 3000 зайнятий, Next.js покаже інший локальний адрес у терміналі.

## Production build

```bash
npm run build
```

## Локальний production-запуск

```bash
npm run start
```

## Змінні середовища

Створити файл `.env.local` у корені проєкту:

```bash
touch .env.local
```

Додати змінні:

```env
OPENAI_API_KEY=
OPENAI_MODEL=
```

На поточному етапі OpenAI не підключений до online-версії, але змінні зарезервовані для майбутнього серверного AI-шару.

## Деплой на Vercel

1. Завантажити проєкт у GitHub.
2. У Vercel створити новий проєкт.
3. Обрати репозиторій з цим Next.js застосунком.
4. Root Directory встановити як `cardio-assistant-online`, якщо деплой іде з монорепозиторію.
5. Build Command залишити `npm run build`.
6. Install Command залишити `npm install`.
7. Output Directory не змінювати.
8. У Vercel додати Environment Variables:
   - `OPENAI_API_KEY`
   - `OPENAI_MODEL`
9. Натиснути Deploy.

## Медичний дисклеймер

Асистент лікаря є допоміжним інструментом для медичних працівників. Результати калькуляторів, опитувальників та AI-згенеровані тексти не замінюють клінічне рішення лікаря.
